Koishi bot GITHUB 模块帮助
#
监听GitHub上的信息。
#
~github add <repo> 订阅库，库需要事先添加过Webhook
~github del <repo> 取消订阅库
~github webhook add <repo> 创建仓库的Webhook用于监听
~github webhook del <repo> 销毁仓库的Webhook
#
~github add/del只能在群或者私聊中触发
~github webhook add/del只能在群临时会话或者私聊中触发
#
使用~github webhook机器人会向你请求发送私人访问令牌，在GitHub中可以从Settings->Developer settings->Personal access tokens中获取。
要求使用的令牌有repo、admin:read_hook和admin_write_hook权限。
机器人不会保存你的令牌，在操作结束之后请立刻撤回你的令牌信息，并删除GitHub上临时创建的令牌。
