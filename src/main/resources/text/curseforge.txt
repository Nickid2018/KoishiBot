Koishi bot CURSEFORGE 模块帮助
#
作用: 获取CurseForge模组信息，使用Modrinth API（可能有一些模组无法搜到）
#
下方所有格式都要以尖括号包括（防止屏蔽）
#
获取模组信息格式:
mod:[模组名称]
模组名称必须和要查询的模组完全一致，否则可能被重定向到错误的模组。
#
查询模组格式:
mod:search:[模组]
mod:search:(页数),(候选数):[模组]
mod:search:(页数),(候选数):[模组]|[查询指令]
结果的候选数量可以由第二种方式指定，但侯选数会限制在30以内。
可以通过第三种方式进一步限制查询，指令类似JQL指令。
#
获取模组文件每个版本URL的格式:
mod:file:[模组名称]
模组名称必须和要查询的模组完全一致，否则可能被重定向到错误的模组。
#
使用例:
<mod:Fabric API>
<mod:search:Fabric>
<mod:file:Fabric API>
#
该功能只能在群聊和私聊中触发，临时会话和陌生人对话不能生效。
#
API说明: https://docs.modrinth.com/api-spec
