Koishi bot CURSEFORGE 模块帮助
#
作用: 获取CurseForge模组信息。
#
下方所有格式都要以尖括号包括（防止屏蔽）
#
获取模组信息格式:
cur:[模组名称]
模组名称必须和要查询的模组完全一致，否则可能被重定向到错误的模组。
#
获取模组文件每个版本URL的格式:
cur:file:[模组名称]
模组名称必须和要查询的模组完全一致，否则可能被重定向到错误的模组。
#
使用例:
<cur:Fabric API>
<cur:file:Fabric API>
#
该功能只能在群聊和私聊中触发，临时会话和陌生人对话不能生效。
#
API说明: https://www.cfwidget.com/
#
